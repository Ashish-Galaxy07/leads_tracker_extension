# 🚀 Leads Tracker Chrome Extension

A modern **Chrome Extension** to save, manage, and organize your important links quickly. Built with **HTML, CSS, and JavaScript**, it features a clean UI, dark mode-ready design, and local storage support. Perfect for productivity and portfolio projects!

---

## 🎯 Features

- ✅ Save URLs manually or directly from the current browser tab  
- ✅ Prevent duplicate links  
- ✅ Delete individual links or clear all at once  
- ✅ Clean, modern, and responsive UI  
- ✅ Dark theme ready  
- ✅ Links open in new tabs for convenience

## 🛠️ How to Use

- Open Chrome → Go to chrome://extensions/
- Enable Developer Mode
- Click Load unpacked and select the project folder
- Use the extension icon to open your Leads Tracker

---

## ⚡ Technologies Used

- HTML
- CSS (modern, responsive, dark mode-ready)
- JavaScript (localStorage + DOM manipulation)
- Chrome Extension API
